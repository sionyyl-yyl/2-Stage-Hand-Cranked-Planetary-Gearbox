# 2-Stage Hand-Cranked Planetary Gearbox

## Overview
This project is a fully 3D printable, hand-cranked 2-stage planetary gearbox.
It is designed for learning mechanical design, CAD modeling, and assembly.

## Status
- CAD: Completed (v1)
- Printing: Not started
- Testing: Not started

## Tools
- SolidWorks
- Bambu P1S (PLA / PETG)

## Structure
- cad/ → SolidWorks files
- stl/ → printable files
- test/ → test logs and results
- images/ → photos and screenshots
- assembly/ → instructions
